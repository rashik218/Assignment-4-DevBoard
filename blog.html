Question-1: What are the different ways to select an element in the DOM?

We can select elements in the DOM in several ways, lke getElementsByTagName, getElementById, getElementsByClassName, querySelector, querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Difference between innerHTML, innerText, and textContent:
innerHtml : We can get and set element visible text content using innerHTML. innerHTML ignore hidden element.
innterText : We can get and set element visible text content using innerText.
textContent : We can get and set element all text content using textContent. textContent also provide hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is a powerful concept of DOM. we can handle event efficiently ushing event delegation. when we need to handle multiple child element through parent element then we use event delegation.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in the DOM (Document Object Model) where an event triggered on a child element bubbles up through its ancestors, eventually reaching the root of the document.

Question-5: How do you create, add, and remove elements using JavaScript?

Create : we can create element using document.createElement('element name')
Add : we can add element using .appendChild(element)
remove : we can remove element.removeChild(element)
